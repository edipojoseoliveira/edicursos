Aprenda a desenvolver sites e aplicações web

Nessa aula teremos uma breve introdução ao que vamos aprender durante o curso de Desenvolvimento de páginas web.

Introdução

Você já teve a vontade ou necessidade de desenvolver um site mas não tinha ideia de como fazer isso? Se sua resposta é sim então esse curso foi feito para você, no curso de Desenvolvimento de páginas web você vai aprender a desenvolver um site completo do início ao fim.

Hoje em dia existem muitas tecnologias para desenvolver páginas web, mas o essencial para desenvolver um site é saber HTML, CSS e JavaScript.

Durante o nosso curso, nós vamos desenvolver um site completo utilizando essas três tecnologias, mas antes vamos ver qual é a função de cada uma delas.

Desenvolvedor front-end e desenvolvedor back-end

Há um tempo atrás, os sites eram desenvolvidos de forma estática, ou seja, os usuários apenas visualizavam as informações das páginas web.

Atualmente os sites tem muito mais funcionalidades, alguns tem jogos, vídeos e músicas, em outros os usuários podem até interagir entre si e em alguns é possível até fazer compras.

Pelo fato dos sites terem mais funcionalidades, as tarefas para desenvolvê-lo ficaram mais complexas e foram divididas em duas áreas. Em uma das áreas é feito o desenvolvimento do front-end onde é criado o visual do site e na outra área é desenvolvido o back-end que é o sistema responsável por gerenciar informações exibidas e obtidas pelo site.

Um desenvolvedor front-end trabalha com várias tecnologias onde as principais são HTML, CSS e JavaScript. Essas linguagens são destinadas ao desenvolvimento do visual dos sites, front-end é tudo que os usuários podem acessar.

Um desenvolvedor back-end trabalha com as tecnologias como Java, C#, PHP, SQL, entre outras. Mas, durante esse curso nosso principal foco vai ser o desenvolvimento front-end das páginas web.

O que eu preciso para desenvolver meu site?

Como vimos nós vamos aprender a desenvolver um site inteiro durante esse curso, e nosso foco principal vai ser o desenvolvimento front-end.

Então precisamos estudar HTML para criar a estrutura das páginas, CSS para estilizar as páginas deixando elas com um visual mais profissional e JavaScript que nos permite adicionar mais funcionalidades ao nosso site.

A seguir, vamos ver a função principal de cada uma dessas tecnologias que iremos estudar durante o curso.

O que é HTML?

HTML é uma linguagem de marcação padrão utilizada no mundo inteiro para desenvolver páginas web. A sigla HTML significa Hypertext Markup Language, que traduzindo para português significa “Linguagem de Marcação de Hipertexto”.

Com o HTML conseguimos definir a estrutura de nossos sites; com ele podemos criar os elementos que compõem o site como tabelas, menus, parágrafos entre outros componentes.

O HTML é a linguagem base da internet, e atualmente todas as páginas da internet que você acessa são criadas com esse padrão. O trabalho do navegador é ler, interpretar o texto escrito em HTML e transformá-lo em um site.

O que é CSS?

A sigla CSS significa Cascading Style Sheets, que traduzindo para português significa “folhas de estilo em cascata”. É uma linguagem que complementa a linguagem HTML.

CSS é a linguagem utilizada para descrever como os componentes do HTML devem ser exibidos no navegador. Como por exemplo, digamos que no nosso site temos uma tabela, essa tabela foi adicionada no site utilizando o HTML, mas para definirmos a cor das linhas dessa tabela precisamos utilizar o CSS.

Com o CSS é possível aplicar estilos aos componentes criados pelo HTML, como alterar a cor, altura, largura, posição na página, bordas e vários outros atributos de todos os elementos do nosso site.

O que é JavaScript?

JavaScript é a linguagem de programação do HTML, com ela você pode programar o comportamento das páginas web.

Com o JavaScript você consegue fazer com que informações sejam atualizadas de forma dinâmica nas páginas, ou seja, com o JavaScript os sites deixam de ser estáticos e passam a ter funcionalidades.

Com esta linguagem conseguimos fazer o front-end se comunicar com o back-end enviando e buscando informações também.

Imagine que você precisa fazer uma página onde o usuário se cadastra no seu site: com o HTML você cria todos os campos e botões na página; com o CSS você deixa sua página mais bonita e com o JavaScript você consegue pegar as informações do usuário e enviar para a aplicação back-end “salvar os dados”.
