O que são operadores matemáticos?

Os operadores matemáticos são os símbolos usados para identificar as operações matemáticas, como por exemplo o operador matemático (+) simboliza a operação de soma na matemática.

No desenvolvimento de software temos os quatro principais operadores matemáticos que são soma (+), subtração (-), multiplicação (*) e divisão (/). Os quais são respectivamente representados pelos símbolos entre parênteses.

Existem algumas diferenças entre os símbolos usados na matemática e os usados em programação, como é o caso do operador de multiplicação, representado pelo asterisco (*) e que na matemática, é representado por um (×). O operador de divisão que é representado por uma barra (/), na matemática é representado pelo seguinte símbolo (÷).

Quando usar os operadores matemáticos?

Os operadores matemáticos são usados quando precisamos realizar algum cálculo em nossos aplicativos, por exemplo, imagine que você está desenvolvendo um aplicativo de venda de livros, você provavelmente vai precisar usar o operador de soma (+) e talvez o operador de multiplicação (*) para calcular o total das compras de cada usuário.

O operador matemático de soma (+) também pode ser usado para concatenar textos, por exemplo, se você tiver uma variável com o nome do usuário e outra variável com o sobrenome, você pode usar o operador (+) para juntar o nome com o sobrenome e guardar o resultado que é o nome completo em outra variável.

Como usar os operadores matemáticos?

Sempre que você for usar um operador matemático é necessário ter os valores que serão usados no cálculo, por exemplo, se o usuário comprou 3 livros e cada livro custa 30 reais você precisará usar o operador de multiplicação (*) para calcular o total da compra.

Nesse exemplo você já tem os valores necessários para realizar o cálculo, agora basta criar uma variável que irá armazenar o resultado da conta 3 * 30, logo a variável receberá o valor de 90 que é o resultado da operação.

Também é necessário dar atenção aos tipos de dados que estão sendo usados: se você fazer a soma de um texto com um número o resultado será a concatenação do texto com o número, por exemplo, no seguinte cálculo “texto” + 7 o resultado será “texto 7”, se você tentar usar um texto com qualquer outro operador matemático além do operador soma (+) ocorrerá um erro.

O termo concatenação é usado para designar a operação de unir o conteúdo de duas ou mais variáveis, no exemplo acima temos uma variável com o conteúdo “texto” e outra com o conteúdo “7”, quando concatenamos as duas temos como resultado a união entre elas formando a palavra “texto 7”.

Quando é feita a soma entre dois números o resultado será um número, no seguinte cálculo 10 + 7 o resultado será 17.
