Aprenda a usar operadores de comparação

Nessa aula vamos aprender o que são os operadores de comparação, qual é sua importância e quando podemos usá-los no desenvolvimento de nossos aplicativos.

Introdução

Nós fazemos diariamente várias comparações, comparamos nossas roupas e calçados para escolher qual usaremos no dia, comparamos preços de produtos para identificar qual oferece mais custo benefício, comparamos músicas para decidir qual ouvir, e várias outras comparações.

No desenvolvimento de nossos aplicativos também haverão momentos em que precisaremos fazer comparações, seja para informar ao usuário qual produto é mais barato ou para saber se o usuário digitou a senha correta no aplicativo.

Essas comparações que fazemos em nossos aplicativos só são possíveis graças aos operadores de comparação.

Nessa aula aprenderemos como funcionam e como usar os operadores de comparação, assim vamos conseguir desenvolver aplicativos cada vez mais inteligentes que conseguem comparar valores e tomar decisões.

O que são operadores de comparação?

Operadores de comparação são símbolos que representam as diferentes operações de comparação que podemos fazer entre dois ou mais valores em nossos aplicativos.

Com esses operadores poderemos saber se um valor é igual a outro, usando o operador Igual (=); podemos verificar se um número é menor que outro usando o operador Menor (<). E também é possível identificar se um valor é maior que outro com o operador Maior (>) entre outros que veremos com mais detalhes a seguir.

Qual a importância dos operadores de comparação?

Eles são de extrema importância para tornar nossos aplicativos mais inteligentes, pois através das comparações o software é capaz de tomar decisões, por exemplo as agendas que avisam quais os compromissos do dia, para isso é usado o operador de comparação que verifica se o dia do compromisso é igual ao dia de hoje.

Com as comparações é possível fazer validações em nossos aplicativos: imagine que em seu aplicativo há um campo onde o usuário digita o mês em que ele nasceu, esse campo não pode aceitar um número maior que 12, para esse tipo de validação usamos os operadores de comparação.

Também usamos os operadores de comparação para deixar nossos sistemas mais seguros, com eles podemos verificar se a senha informada pelo usuário é a mesma senha que foi cadastrada.

Quando usá-los?

Você pode usar os operadores de comparação quando tiver dois ou mais valores e quer compará-los, por exemplo, imagine que você tem uma lista de produtos e precisa encontrar o mais barato, nesse caso você vai precisar comparar o valor de cada produto para descobrir qual é o menor.

Você também pode usar os operadores de comparação para criar validações, vamos supor que no seu aplicativo o usuário vai precisar fazer login para acessar as informações, nesse caso quando o usuário informa a senha seu aplicativo precisa verificar se a senha digitada é igual a senha gravada no cadastro do usuário.

Como usar os operadores de comparação?

Suponha que no seu aplicativo há dois produtos e você precisa indicar para o usuário qual é o produto mais barato, então você poderia fazer a comparação da seguinte forma (preço do primeiro produto < preço do segundo produto).

Se o resultado da comparação feita anteriormente for Verdadeiro significa que o preço do primeiro produto é menor que o preço do segundo, se o resultado for Falso então o preço do segundo produto é menor.

No caso de você precisar validar se uma senha é igual a outra, o operador de comparação “Igual” que é representado pelo seguinte símbolo (=) seria o mais indicado para resolver seu problema.

Você poderia usar a seguinte expressão (senha digitada = senha cadastrada), se o resultado for Verdadeiro significa que a senha digitada é igual a senha cadastrada, e se o resultado for Falso significa que as senhas são diferentes.

O resultado do uso de um operador de comparação sempre será Verdadeiro ou Falso, no exemplo anterior quando as senhas forem diferentes o resultado será Falso, caso as senhas realmente sejam iguais então o resultado será Verdadeiro.

Na maioria das linguagens de programação o valor Verdadeiro é representado por True que é verdadeiro em inglês e o valor Falso é representado por False, mas como nós estamos usando o App Inventor traduzido para português ele vai exibir o resultado das comparações em português também.

Uma coisa muito importante ao comparar textos é que uma letra maiúscula é diferente de uma minúscula, veja a seguinte expressão: (“Texto” = “texto”), nesse exemplo o resultado será Falso pois o primeiro valor tem um “T” maiúsculo enquanto no segundo usa o “t” minúsculo.

O App Inventor disponibiliza todos os operadores de comparação que você pode usar em qualquer linguagem de programação.

Veja a seguir uma lista com todos os operadores de comparação do App Inventor.

Símbolo Nome Função Exemplo
= Igual Usado para verificar se um valor é igual a outro 3 = 5 (False)
> Maior Usado para verificar se um valor é maior que outro 3 > 5 (False)
< Menor Usado para verificar se um valor é menor que outro 3 < 5 (True)
