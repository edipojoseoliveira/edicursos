Operadores de comparação

Seja bem-vindo(a) a nossa quinta aula do curso de lógica de programação com App Inventor, na aula anterior nós falamos sobre os operadores matemáticos que podemos usar em nossos aplicativos.

Vimos que os operadores matemáticos são usados para fazer cálculos entre números, e o resultado do uso de uma operação matemática sempre será um número. Por exemplo, o resultado da soma entre os número 10 e 5 será 15.

Um operador de comparação é usado para comparar qualquer valor, não apenas números. Podemos usá-los em casos onde precisamos saber se uma variável tem o mesmo valor que outra, ou se tem o valor maior que outra, e várias outras comparações, o resultado do uso de uma operação de comparação sempre será Verdadeiro (True) ou Falso (False), por exemplo, quando comparamos se uma variável é igual a outra o resultado dessa comparação vai ser Verdadeiro (True) se elas realmente são iguais ou Falso (False) caso elas sejam diferentes.

Durante o desenvolvimento de nossos aplicativos, em vários momentos nós precisamos comparar valores, por exemplo, imagine que precisamos acessar nossa conta do facebook, precisamos lembrar do login e da senha, mas como o Facebook sabe que digitei a senha certa e o login certo? Provavelmente ele faz uma comparação entre a senha digitada e a senha que já esta cadastrada no site.

Percebeu como os operadores de comparação são importantes? Nós temos vários operadores de comparação disponíveis para usar, veja abaixo uma lista dos operadores de comparação que o App Inventor disponibiliza.

Operadores de comparação disponíveis no App Inventor

Símbolo Nome Função Exemplos
= Igual Usado para verificar se um valor é igual a outro 3 = 5 (False)
≠ Diferente Usado para verificar se um valor é diferente de outro 3 ≠ 5 (True)
< Menor Usado para verificar se um valor é menor que outro 3 < 5 (True)
≤ Menor ou igual Usado para verificar se um valor é menor ou igual a outro 3 ≤ 5 (True)
> Maior Usado para verificar se um valor é maior que outro 3 > 5 (False)
≥ Maior ou igual Usado para verificar se um valor é maior ou igual a outro 3 ≥ 5 (False)
