Estruturas de repetição

Seja bem-vindo(a) a nossa sétima aula do curso de Lógica de Programação com App Inventor, nessa aula vamos estudar as estruturas de repetição.

As estruturas de repetição são usadas quando precisamos que um determinado comando seja executado várias vezes seguidas.

Imagine que você precisa desenvolver um aplicativo que a cada 1 hora verifica a temperatura ambiente e mostra essa informação para o usuário, nesse caso você precisa repetir o comando de consultar a temperatura a cada 1 hora, usando uma estrutura de repetição você conseguiria fazer esse aplicativo de forma muito fácil. Esse foi só um exemplo de uma situação onde poderíamos resolver o problema usando estruturas de repetição.

No App Inventor existem três estruturas de repetição, são elas:

  1. Para cada número repete, mais conhecida como (for)
  2. Para cada elemento na lista fazer, mais conhecida como (for each)
  3. Enquanto testar repete, mais conhecida como (while)

Estrutura de repetição (for)

Vamos começar estudando a estrutura for. Nós usamos essa estrutura quando precisamos repetir um ou vários comandos por uma quantidade determinada de vezes.

Imagine que você quer que seu aplicativo escreva na tela os número de 1 até 5, você pode fazer isso de duas formas, a primeira é você usar o comando de imprimir um número na tela 5 vezes seguidas.

A segunda forma é você usar a estrutura de repetição que faz repetir por 5 vezes o comando de imprimir um número na tela.

Assim você precisa usar apenas uma vez o comando de imprimir um número na tela e informar para a estrutura de repetição que você deseja repetir o comando por 5 vezes.

Como você pode ver nesse exemplo, quando não usamos a estrutura de repetição nós temos que repetir muitos blocos, já quanto usamos a estrutura de repetição a quantidade de blocos usados é menor.

Agora imagine se em vez de ter que imprimir os número de 1 a 5 você tivesse que imprimir os números de 1 a 100. Com a estrutura de repetição você só precisa alterar a quantidade de vezes que ela deve repetir, no lugar de 5 digitar 100, já se você não tivesse a estrutura de repetição você teria que copiar 100 vezes os mesmos blocos.

Essa primeira estrutura de repetição tem o objetivo de repetir por uma quantidade definida de vezes os comandos que quisermos, vamos ver agora a próxima estrutura de repetição.

Estrutura de repetição (for each)

A estrutura de repetição for each é usada quando temos uma lista e precisamos percorrer todos os seus itens.

Imagine que temos em nosso aplicativo uma lista de compras e agora precisamos mostrar todos os itens dessa lista na tela para o usuário.

Nesse caso nós usamos a estrutura de repetição for each, para essa estrutura de repetição funcionar precisamos informar para ela qual lista deve ser percorrida. Então para cada item da lista será executado o comando para imprimi-lo na tela.

Veja como ficaria a organização dos blocos.

Primeiro criamos uma variável que guardará a lista de compras com os itens: Arroz, Feijão, Tomate, Alface, Maçã, Banana.

Ao clicar em um botão da tela usamos a estrutura de repetição para percorrer todos os itens da lista de compras e imprimimos na tela item por item.

Essa segunda estrutura de repetição tem o objetivo de percorrer todos os itens de qualquer lista que tivermos, vamos ver agora como funciona a estrutura de repetição while.

Estrutura de repetição (while)

A estrutura de repetição while é usada quando temos que repetir alguns comandos mas não sabemos exatamente quantas vezes precisamos repeti-los, a quantidade de vezes que será executado vai depender de uma condição.

Imagine que você precisa agora criar um despertador, o aplicativo deve saber que horário vai disparar o alarme, mas não sabe quantas vezes o alarme vai tocar, pois vai depender de quando o usuário clicar no botão desligar do despertador.

Enquanto o usuário não clicar no botão desligar alarme, ele deve continuar tocando. Perceba que nesse caso você não sabe exatamente quantas vezes o alarme deve tocar, mas sabe que enquanto o botão desligar não for clicado o alarme deve continuar tocando repetidamente.
