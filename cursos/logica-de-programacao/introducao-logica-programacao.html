Introdução

Você quer desenvolver seus próprios aplicativos para celular mas não sabe por onde começar?

Estudar lógica de programação é o primeiro passo para iniciar uma carreira na área de desenvolvimento de software ou até mesmo para melhorar suas habilidades como programador(a).

Nessa aula aprenderemos o que é lógica de programação, sua importância, onde podemos usá-la e vamos conferir como ela pode te ajudar no desenvolvimento de seus aplicativos.

O que é lógica de programação?

De uma maneira mais formal, a lógica de programação pode ser definida da seguinte forma: “Lógica de programação é a técnica de desenvolver sequências lógicas para atingir um determinado objetivo.” (JOAQUIM, 2014, P. 1).

Ela está relacionada com a maneira que você organiza seus pensamentos, não é apenas saber escrever códigos de programação.

Com ela você aprende primeiro a organizar suas ideias de forma lógica, para depois transformá-las em códigos de programação que o computador pode executar para realizar uma tarefa.

Imagine que precisamos fazer um programa para controlar o estoque dos produtos de uma loja.

Uma das funcionalidades que esse programa deve ter é “exibir uma mensagem avisando o usuário quando um produto estiver acabado no estoque”.

Para desenvolver essa funcionalidade primeiro precisamos criar a lógica que passaremos para o computador através de uma linguagem de programação.

Por exemplo, a minha lógica seria a seguinte:

  1. Preciso obter a quantidade atual do produto no estoque.
  2. Verificar se a quantidade do produto é maior que zero.
  3. Se a quantidade for zero então vou exibir uma mensagem para o usuário informando que o produto está sem quantidade em estoque.

Pronto, minha lógica já está desenvolvida com apenas três etapas, agora preciso apenas traduzir essa sequência de passos para uma linguagem de programação, que é a linguagem que o computador conseguirá interpretar.

É interessante lembrar que é possível desenvolver várias lógicas para atingir um mesmo objetivo, como nesse caso do controle de estoque, você pode ter pensado em outros passos para atingir o mesmo resultado.

No início pode ser difícil conseguir pensar de forma lógica para desenvolver um software, mas com o passar do tempo e com a prática esse processo vai se tornando natural.

O que é algoritmo?

Primeiro vamos ver a definição da palavra algoritmo no Dicionário Priberam: “Sequência finita de instruções não ambíguas utilizadas para resolver um problema ou fazer um cálculo”.

Você já fez um bolo de chocolate? Imagine os algoritmos como sendo receitas de bolo, na verdade uma receita de bolo é um algoritmo, pois ela é um conjunto de regras ou passos, que estão organizados de forma correta para atingir um objetivo.

De forma resumida, algoritmo é um conjunto de passos ordenados e finitos, que ao serem executados conseguem chegar a um objetivo ou executar uma tarefa.

Algoritmos da forma que você conhece

Todos nós usamos algoritmos a quase todo momento sem perceber, por exemplo, quando você vai fazer uma ligação telefônica para alguém.

Na hora de fazer a ligação você segue alguns passos, esses passos são ordenados e finitos como um algoritmo.

Veja abaixo o algoritmo que eu normalmente uso para fazer uma ligação.

  1. Pegar o celular
  2. Digitar a senha
  3. Abrir o aplicativo de chamadas
  4. Digitar o número da pessoa que quero ligar
  5. Clicar no botão chamar

Perceba que se trata de um algoritmo bem simples, mas com ele eu consigo atingir meu objetivo de ligar para alguém.

Talvez você siga uma outra sequência de passos, mas independentemente do algoritmo usado, você vai conseguir atingir seu objetivo.

Qual a relação entre lógica de programação e algoritmos?

Como vimos anteriormente, lógica de programação é a técnica de desenvolver sequências lógicas para atingir um determinado objetivo.

E algoritmo é uma sequência lógica e finita de passos usada para atingir um determinado objetivo.

Ou seja, a lógica de programação é a técnica de desenvolver algoritmos. Percebeu a relação entre esses dois assuntos?

Com uma boa lógica de programação você vai conseguir desenvolver o melhor algoritmo para atingir seu objetivo, e após desenvolver o algoritmo, basta adaptá-lo para uma linguagem de programação a fim de produzir um software.

Porque aprender lógica de programação?

A tecnologia está dominando muitas áreas das nossas vidas, e isso é apenas o começo.

Estudar lógica de programação não é algo apenas para pessoas que desejam se tornar desenvolvedoras de software.

Esse assunto pode ser estudado por todas as pessoas, pois independente da área em que você trabalha, a tecnologia faz ou fará parte do seu cotidiano, saber como ela funciona é a maneira mais fácil de se adaptar a essas mudanças.

Onde posso usá-la?

Em todo o processo de desenvolvimento de um software, não importa se você vai criar um aplicativo simples, desenvolver um game, um sistema sofisticado para gestão de empresas, ou um site e-commerce, a lógica de programação sempre será usada.

Para desenvolver qualquer programa, primeiro você precisa pensar em todo seu funcionamento, organizar as ideias de forma lógica, e após isso transformar suas ideias em códigos de programação que o computador vai executar.

Em todos os softwares que você for desenvolver vai ser necessário saber usar a lógica de programação para conseguir fazer tudo funcionar.

O que posso fazer após aprender a programar?

Você pode começar desenvolvendo coisas para você mesmo, como por exemplo, um aplicativo para escrever suas anotações, ou um game para jogar com seus amigos.

Com o passar do tempo você vai ter mais conhecimento e poderá desenvolver softwares profissionais, como sites para expor seus produtos ou aplicativos para sua empresa.

Pode ter certeza que você vai conseguir desenvolver qualquer software, há uma frase que eu gosto bastante: Tudo que você é capaz de imaginar, você é capaz de realizar. (Walt Disney)

Em programação é mais ou menos assim, tudo que você consegue imaginar, você consegue desenvolver.

Conclusão

Nessa aula aprendemos a importância que a lógica de programação tem na carreira de um(a) programador(a).

É com ela que todo o desenvolvimento de um software se torna possível, após dominar esse assunto você vai conseguir desenvolver o software que quiser.

No entanto é muito importante se dedicar a esse curso, pois ele servirá de base para tudo que você vai estudar sobre programação no futuro.
