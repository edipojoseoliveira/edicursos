Introdução a lógica de programação

Seja bem-vindo(a) a nossa primeira aula do curso de lógica de programação com App Inventor.

Muitas pessoas querem aprender a programar e desenvolver seus próprios aplicativos e sistemas.

O primeiro passo é estudar lógica de programação, pois ela é a base de tudo em programação, com a lógica de programação aprendemos a escrever códigos de forma organizada para o computador interpretar.

No começo é normal surgir a dúvida sobre qual linguagem de programação aprender primeiro, mas a linguagem não é o item mais importante no início, pois a lógica vai ser a mesma em todas as linguagens.

Lógica de Programação

Lógica de programação é a técnica de desenvolver uma sequência de passos para atingir um determinado objetivo.

Essa sequência de passos deve estar organizada para ser executada pelo computador.

É importante lembrar que o computador não é inteligente para saber o que precisa ser feito, ele precisa de instruções objetivas, detalhadas e na ordem correta para executar uma tarefa.

Algoritmo

A sequência de passos que precisamos desenvolver é chamada de algoritmo. O algoritmo não é usado só em programação, na verdade usamos algoritmos em quase todo momento.

Vamos ver um exemplo de algoritmo para trocar uma lâmpada queimada:

Início

  1. Pegar uma escada
  2. Colocar a escada embaixo da lâmpada
  3. Subir na escada
  4. Retirar a lâmpada queimada
  5. Descer da escada
  6. Pegar a lâmpada nova
  7. Subir na escada
  8. Colocar a lâmpada nova
  9. Descer da escada

Fim

Esse é um exemplo simples de algoritmo, nele podemos ver que existe uma sequência de passos ordenados, que devem ser executados para atingir nosso objetivo, que é trocar a lâmpada queimada.

A diferença entre o algoritmo que criamos no exemplo e o algoritmo que usamos na programação é a linguagem. Em programação normalmente usamos comandos em Inglês e temos um conjunto limitado de palavras disponíveis.
