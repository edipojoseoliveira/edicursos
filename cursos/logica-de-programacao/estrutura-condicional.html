Estrutura condicional (IF e ELSE)

Seja bem-vindo(a) a nossa sexta aula do curso de lógica de programação com App Inventor, nesta aula vamos falar sobre estrutura condicional (IF e ELSE).

Para que você consiga entender bem o conteúdo dessa aula, é necessário que tenha participado da aula anterior, onde estudamos os operadores de comparação, pois em uma estrutura condicional você vai precisar usá-los, caso não tenha visto a aula anterior clique aqui para acessá-la.

O que é uma estrutura condicional?

Essa provavelmente foi sua primeira dúvida ao iniciar essa aula. O nome parece difícil “estrutura condicional”, mas o que ela representa é algo simples, e nós já usamos ela todos os dias, sem nem perceber.

Vamos ver um exemplo de estrutura condicional que você já deve ter usado. Imagine que você precisa atravessar uma rua, mas está com medo de ser atropelado(a), você pode usar a seguinte estrutura condicional: 

Se não estiver vindo carros da direita e não estiver vindo carros da esquerda então
    Eu posso atravessar.
Senão
    Eu tenho que esperar.

Essa é uma estrutura condicional, parece bem simples não é?

Vamos ver mais um exemplo que você já deve ter usado também. Imagine que você está escolhendo uma roupa para usar amanhã, mas está em dúvida em usar uma blusa ou uma camiseta, você pode usar a seguinte estrutura condicional: 

Se estiver frio então
    Eu uso uma blusa
Senão
    Eu uso uma camiseta.

Viu só como a estrutura condicional é fácil, e você já usa ela em vários momentos do seu dia a dia.

Você percebeu alguma semelhança entre os dois exemplos acima?

Toda estrutura condicional começa com a palavra Se, depois vem a condição, se a condição for atendida então a próxima linha é executada, caso a condição não seja atendida a linha que será executada é a que está abaixo da palavra Senão.

Em programação a lógica é exatamente a mesma, a diferença é que para criar as condições nós não vamos apenas escrever, vamos ter que usar os operadores de comparação para fazer as comparações necessárias.

Basicamente, a estrutura condicional permite que os programas possam verificar quando um determinado comando deve ser executado e quando não deve ser executado.
